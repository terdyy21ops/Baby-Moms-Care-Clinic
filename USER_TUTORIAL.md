### ğŸ“‹ Complete Appointment Process

1. Mother logs into account

7. Appointment created with status: **ğŸŸ¡ PENDING**

1. Doctor receives email notification
2. In-app notification appears (red badge on bell icon)
3. Doctor logs into account
4. Sees pending request on dashboard
5. Clicks to view appointment details

**STEP 3: Doctor Reviews and Approves**
1. Doctor reviews patient information
2. Checks reason for visit
3. Verifies schedule availability
4. Doctor has three options:
   - **Approve**: Confirms appointment
   - **Reject**: Declines with reason
   - **Reschedule**: Suggests alternative time

**If Doctor Approves:**
1. Doctor clicks "Approve" button
2. Can add confirmation notes
3. Clicks "Confirm Approval"
4. Status changes to: **ğŸŸ¢ CONFIRMED**
5. Mother receives approval notification
6. Appointment added to both calendars

**If Doctor Rejects:**
1. Doctor clicks "Reject" button
2. Must enter reason for rejection
3. Can suggest alternative doctor/time
4. Status changes to: **ğŸ”´ CANCELLED**
5. Mother receives rejection notification with reason
6. Mother can book with another doctor

**STEP 4: Appointment Reminders**
1. **24 Hours Before**: Both parties receive reminder email
2. **2 Hours Before**: In-app notification sent
3. **30 Minutes Before**: Final reminder (optional)

**STEP 5: Appointment Day**
1. Mother attends appointment at scheduled time
2. Doctor conducts consultation
3. Doctor provides care and advice

**STEP 6: Doctor Marks as Completed**
1. After consultation, doctor logs in
2. Goes to appointment details
3. Clicks "Complete Appointment"
4. Fills in:
   - Diagnosis
   - Prescription
   - Recommendations
   - Follow-up required (Yes/No)
   - Next visit date (if needed)
   - Doctor's notes
5. Clicks "Mark as Completed"
6. Status changes to: **ğŸ”µ COMPLETED**

**STEP 7: Post-Appointment**
1. Mother receives completion notification
2. Mother can view consultation notes (if shared)
3. Mother can download appointment receipt
4. Mother can rate the doctor (optional)
5. If follow-up needed, mother books next appointment

**STEP 8: Admin Oversight**
1. Admin can view all appointments in system
2. Monitor appointment statistics
3. Handle any issues or disputes
4. Generate reports for analysis
